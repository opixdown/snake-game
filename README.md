# 🐍 Snake Game (C + SDL)

## 📌 Overview  
Welcome to the Snake Game! This is a classic arcade style game built in C using SDL (Simple DirectMedia Layer). The objective is simple control the snake, eat food to grow, and avoid collisions. This project helped us explore low level graphics programming and real time game logic.

---

## 🚀 Features  
- Classic snake gameplay  
- Smooth keyboard controls  
- Score tracking system  
- Game over screen  
- Built using SDL for graphics and input handling  

---

## 🧠 Technologies Used  
- C Programming  
- SDL (Simple DirectMedia Layer)  

**About SDL:**  
SDL is a cross-platform multimedia library that provides low-level access to graphics, audio, and input devices. It is widely used for game development and interactive applications.

**Why SDL?**  
- Lightweight and easy to learn  
- Opensource (zlib license)  
- Cross platform support  
---

## ⚙️ Setup Instructions  

1. Extract the provided ZIP folder  
2. Copy the folder named "pacmanimages" to your C drive  
3. Open the project in Microsoft Visual Studio  
4. Add all required libraries from the provided library folder to the project  
5. Build the project and run the game  

---

## 🎮 How to Play  

### ▶️ Launch the Game  
- Run the project to see the welcome screen  

### 🕹️ Start Playing  
- Click on the Play button  
- Use arrow keys to control the snake  
- Eat the food to grow longer  

### ⚠️ Game Rules  
- Avoid hitting the walls  
- Avoid colliding with your own body  

---

## 💀 Game Over  
- The game ends when the snake crashes  
- Your final score will be displayed on the end screen  

---

## 🎯 Learning Outcomes  
- Understanding of SDL and graphics rendering  
- Handling real time input (keyboard controls)  
- Game loop implementation  
- Collision detection and game logic
